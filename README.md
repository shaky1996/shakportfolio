# Shak Yuldashev Portfolio

A responsive React portfolio built with Vite. The interface follows the supplied
dark, technical mockup and is organized into reusable components for the
navigation, hero, about, skills, projects, certifications, and contact sections.

## Run locally

```bash
npm install
npm run dev
```

Create a production build with:

```bash
npm run build
```

Portfolio content and external URLs are centralized in
`src/data/portfolio.js`.
